# 视觉设计规范（v1）

## 字体与尺寸
- 基础字体：16px（最小 14px）
- 表格字体：16px
- 标签文字：16px

## 输入控件
- 高度：32px
- 内边距：6px 12px
- 圆角：4px
- 字体：16px

## 按钮
- 主按钮高度：32px
- 次按钮高度：28px
- 圆角：4px
- 阴影：
  - 默认：0 1px 2px rgba(0,0,0,0.12)
  - Hover：0 4px 12px rgba(0,0,0,0.18)
  - Active：0 6px 18px rgba(0,0,0,0.22)

## 搜索区域
- 行高：56px
- 间距：20px
- 区域内边距：16px
- “全选”行：高度 56px，对齐表格选择列

## 日期组件
- 统一宽度：300px

## 交互状态
- 输入 Focus 高亮：绿色外发光 `rgba(76,175,80,0.25)`

## 响应式
- 小屏（≤768px）：搜索区域间距降至 12px；按钮字体 14px

## 技术落地
- 全局样式文件：`src/styles/global.css`
- 引入位置：`src/main.ts`
- 表格页面统一结构：Header（操作）→ 搜索行 → 全选行 → 表格

## 导出规范
- 文件格式：xlsx
- 列宽策略：按内容自适应，文本列宽约 等比扩大，最小 18；数字列 12
- 全选导出范围：所有数据（不受分页与筛选限制）

## 测试
- 浏览器：Edge 最新版（项目运行环境）、Chrome 最新版
- 设备：1920×1080 桌面、1280×800 笔记本、375×667 移动端模拟
